# كمبروسر الرياض المطور | Kompressor Riyadh

المؤسسة الرائدة بالرياض لشفط وتسليك المجاري بالضغط والكمبروسر الحديث.

## المميزات
- دعم كامل للغة العربية (RTL)
- تصميم عصري باستخدام TailwindCSS و Framer Motion
- متوافق مع جميع الأجهزة (Responsive Design)
- نظام طوارئ 24 ساعة مع تكامل واتساب

## التقنيات المستخدمة
- Next.js 14
- TypeScript
- TailwindCSS
- Framer Motion
- Lucide React

## التشغيل
```bash
npm install
npm run dev
```
